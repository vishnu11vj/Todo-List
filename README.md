# ✅ Todo List App

A clean and minimal task manager with a warm paper-like aesthetic, built with HTML, CSS, and JavaScript.

## Live Demo
Open `index.html` in any browser — no installation needed.

## Features
- Add, complete, and delete tasks
- Filter by All / Active / Completed
- Task count tracker
- Data saved in localStorage (persists after refresh)
- Smooth animations on task entry

## Built With
- HTML5
- CSS3
- Vanilla JavaScript
- localStorage API

## How to Run
Clone the repo and open `index.html` in your browser.
